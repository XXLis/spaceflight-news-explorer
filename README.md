# 🚀 Spaceflight News API Explorer

Een interactieve webapplicatie die de laatste 10 ruimtevaartblogs ophaalt en weergeeft met behulp van de **Spaceflight News API**.

## 🛠️ Technologieën
* **JavaScript (ES6+)**: Async/Await, Fetch API, DOM Manipulation.
* **HTML5 & CSS3**: Responsive Grid Layout.
* **API**: [Spaceflight News API v4](https://api.spaceflightnewsapi.net/v4/docs/)

## 📋 Functies
- Haalt live data op van externe servers.
- Foutafhandeling (Error Handling) voor netwerkproblemen.
- Responsive design dat werkt op mobiel en desktop.
- Clean code architectuur met gescheiden functies.

## 🚀 Hoe te gebruiken
1. Clone deze repository.
2. Open `index.html` in je browser.
3. Bekijk de console (F12) om de ruwe data te zien, of bekijk de visuele kaarten op de pagina.
