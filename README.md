# 🚀 Spaceflight News API Explorer

Een interactieve webapplicatie die de laatste 10 ruimtevaartblogs ophaalt en weergeeft met behulp van de **Spaceflight News API**.

## 🔗 Live Demo
Bekijk het project hier: [https://xxlis.github.io/spaceflight-news-explorer/](https://xxlis.github.io/spaceflight-news-explorer/)

## 🛠️ Technologieën
* **JavaScript (ES6+)**: Gebruik van Fetch API en Async/Await voor data-ophaling.
* **HTML5 & CSS3**: Een responsive layout met CSS Grid.
* **API**: [Spaceflight News API v4](https://api.spaceflightnewsapi.net/v4/docs/).

## 📋 Functies & Opdracht
Dit project is gebouwd als onderdeel van een BitAcademy opdracht. De focus lag op:
- **Data fetching**: Het ophalen van 10 blogposts van een externe API.
- **Console Output**: Zoals gevraagd in de opdracht, worden de specifieke datapunten (`id`, `title`, `summary`, etc.) ook gelogd in de browser console voor inspectie.
- **Error Handling**: Het opvangen van fouten tijdens het laden van de data.

## 💻 Hoe te bekijken
1. Open de [Live Demo](https://xxlis.github.io/spaceflight-news-explorer/).
2. De blogs worden automatisch geladen op de pagina.
3. **Voor developers**: Druk op `F12` of `Rechtermuisknop > Inspecteren > Console` om de ruwe data-objecten te bekijken.

## 🚀 Installatie (Lokaal)
1. Clone deze repository: `git clone https://github.com/xxlis/spaceflight-news-explorer.git`
2. Open `index.html` in je favoriete browser.
